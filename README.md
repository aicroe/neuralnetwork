# Neural network

A simple implementation of a Neural Network using OOP concepts.

## Scripts

### Install

```bash
make install
```

### Test

```bash
make test
```

### Dev Mode

```bash
make start
```

### Clean

Uninstall the package.

```bash
make clean
```

## Virtual Environment

Recommended setup.

Create the virtual environment.
```bash
python -m venv .venv
```

Activate the environment.
```bash
source .venv/bin/activate
```

Deactivate the environment.
```bash
deactivate
```
